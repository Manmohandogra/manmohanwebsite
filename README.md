# Manmohan Dogra — Personal Website

Repository for the personal website of Manmohan Dogra, Tabla Virtuoso.

## Structure

```
data/           # Structured JSON data
  artist.json   # Bio, performances, awards, education
  contact.json  # Socials, email, WhatsApp (private)
  photos.json   # Photo manifest
content/        # Markdown content
  about.md      # Full artist biography
*.jpg / *.JPG   # Performance and portrait photographs
```
